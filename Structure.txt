🧠 Why This Is Important

In production, orchestrators and load balancers continuously check if a service is healthy.

Without health checks:
Client
   ↓
❌ Down Server

With health checks:
           Load Balancer
           /           \
      Healthy        Unhealthy
       Server           Server
         ✔               ❌

Benefits:
	•	✅ Automatic failure detection
	•	✅ Better uptime
	•	✅ Faster recovery
	•	✅ Improved monitoring
